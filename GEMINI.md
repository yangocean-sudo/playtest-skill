# Playtest Instructions For Gemini-Style Agents

When asked to evaluate product usability as a real user, follow the Playtest workflow from `AGENTS.md`.

Use the local tool equivalents available in the environment:

- browser automation for web products
- simulator automation for mobile products
- screenshots for evidence
- file editing tools for `docs/playtest/<date>/`

Do not run against production or shared customer data. Do not skip the human scope checkpoint after `test-plan.md`.

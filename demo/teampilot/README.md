# TeamPilot Demo

TeamPilot is a deliberately small SaaS-style demo app for showing what the Playtest skill catches.

It is intentionally not perfect. The product technically works, but several common UX problems are present:

- weak empty state after workspace creation
- hidden invite action
- unclear role-management jargon
- mobile navigation that hides the next important action

Run it from the repo root:

```bash
npm run serve:demo
```

Then open:

```text
http://localhost:4173
```
